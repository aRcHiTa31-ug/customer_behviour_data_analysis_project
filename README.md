# customer_behviour_data_analysis_project
This project involves cleaning and preprocessing data using Python (Pandas and NumPy), storing the cleaned data in a MySQL database, and analyzing it using SQL queries. The data is then connected to Power BI through ODBC to create interactive dashboards and visualizations for meaningful insights and decision-making.
